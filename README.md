# 辦公室飲料訂購試算系統 v4

手繪插畫風單頁系統，可直接上傳 GitHub Pages 使用。

## 主要功能

- 輸入店家、日期
- 輸入同仁姓名、品項、單價、數量
- 輸入「平台最後實付總額」
- 系統自動反推差額：
  - 最後實付總額 > 飲料原價總額：多出來的費用平均加回每個人
  - 最後實付總額 < 飲料原價總額：折扣平均扣除給每個人
- 可勾選是否參與分攤
- 試算結果只顯示「姓名：xx元」
- 關閉網頁後暫存資料仍會保留
- 結清後存入歷史紀錄並清空目前畫面
- 可載入、複製、刪除歷史紀錄
- 可匯出 CSV、列印 / 存 PDF

## 使用方式

直接開啟 `index.html` 即可使用。

若要上傳 GitHub Pages，將整個資料夾內容放到 repository，並啟用 Pages。


## v5 更新

- 新增「常用人員建檔」：可把常訂飲料的同仁存在瀏覽器中。
- 訂購明細姓名欄支援下拉建議與「選」按鈕快速帶入。
- 點常用人員膠囊會自動填入第一個空白姓名欄；若沒有空白列，會自動新增一列。
- 不在人員名單內的人仍可直接手動輸入。
- 右鍵點常用人員膠囊可移除該人員；或使用「清空人員名單」。
